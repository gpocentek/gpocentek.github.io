Bridges!

Configuration samples to setup bridges on Linux systems

Linux Bridge

Setup of a br0 with a eth0 port.

  • On RHEL 7:

    # yum install bridge-utils
# cat /etc/sysconfig/network-scripts/ifcfg-eth0
DEVICE=eth0
TYPE=Ethernet
BOOTPROTO=none
ONBOOT=yes
NM_CONTROLLERD=no
BRIDGE=br0
# cat /etc/sysconfig/network-scripts/ifcfg-br0
DEVICE=br0
TYPE=Bridge
ONBOOT=yes
BOOTPROTO=dhcp
NM_CONTROLLED=no
DELAY=0

  • On Debian:

    # apt-get install bridge-utils
# cat /etc/network/interfaces
...
auto br0
iface br0 inet dhcp
    bridge_ports eth0
